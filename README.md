# 파이썬 기초 연습문제

앞서 배운 5가지 주제에 대해 각각 5개의 연습문제를 준비했습니다.
직접 코드를 작성하며 풀어보세요!

---

## 1. 변수와 자료형 (01_hello_variables.py 관련)

1. **자기소개 변수 만들기**
   - 이름(`name`), 나이(`age`), 사는 곳(`city`)을 변수에 저장하고, f-string을 사용하여 "저는 [사는 곳]에 사는 [나이]살 [이름]입니다."라고 출력하세요.

2. **사각형 넓이 구하기**
   - 가로(`width`)가 20, 세로(`height`)가 30인 변수를 만들고, 넓이(`area`)를 계산하여 출력하세요.

3. **하루는 몇 초인가?**
   - 하루는 24시간, 1시간은 60분, 1분은 60초입니다. 연산을 통해 하루가 총 몇 초인지 계산하여 변수에 담고 출력하세요.

4. **단위 변환기 (인치 -> 센티미터)**
   - `inch` 변수에 5를 저장하고, 이를 cm로 변환(1 inch = 2.54 cm)하여 출력하세요.

5. **변수 값 서로 바꾸기 (Swap)**
   - `a = 10`, `b = 20`이 있을 때, 두 변수의 값을 서로 바꾸어 `a`가 20, `b`가 10이 되도록 코드를 작성하고 출력하세요.

---

## 2. 조건문 (02_control_flow.py 관련)

1. **홀수/짝수 판별기**
   - 숫자 하나를 변수 `num`에 저장하고, 이 숫자가 짝수이면 "짝수", 홀수이면 "홀수"를 출력하세요.

2. **성인 판독기**
   - 나이(`age`) 변수의 값에 따라, 20세 이상이면 "성인입니다", 20세 미만이면 "미성년자입니다"를 출력하세요.

3. **큰 수 찾기**
   - 두 숫자 `num1`, `num2`가 있을 때, 둘 중 더 큰 숫자를 찾아 "더 큰 수는 O입니다"라고 출력하세요. (같으면 "두 수는 같습니다" 출력)

4. **학점 계산기 2**
   - 점수(`score`)가 90점 이상이면 "A", 80점 이상이면 "B", 70점 이상이면 "C", 그 외에는 "F"를 출력하는 코드를 작성하세요.

5. **3의 배수인지 확인하기**
   - 어떤 숫자 `n`이 3의 배수이면서 동시에 5의 배수라면 "3과 5의 공배수", 아니면 "해당 없음"을 출력하세요.

---

## 3. 반복문 (03_loops.py 관련)

1. **1부터 10까지 합 구하기**
   - `for`문을 사용하여 1부터 10까지의 모든 숫자를 더한 결과를 출력하세요. (답: 55)

2. **구구단 7단 출력**
   - `for`문을 사용하여 구구단 7단(7x1 ~ 7x9)을 출력하세요.

3. **카운트다운**
   - `while`문을 사용하여 10부터 1까지 거꾸로 출력하고, 마지막에 "Happy New Year!"를 출력하세요.

4. **100 이하의 짝수만 더하기**
   - 1부터 100까지의 숫자 중 **짝수**들의 합만 구하여 출력하세요. (`range`의 step 기능을 활용하거나 `if`문을 사용하세요)

5. **별 찍기**
   - 반복문을 사용하여 아래와 같은 모양을 출력하세요.
     ```
     *
     **
     ***
     ****
     *****
     ```

## 4. 함수 (04_functions.py 관련)

1. **두 수의 합 함수**
   - 두 개의 숫자를 입력받아 그 합을 **반환(return)**하는 함수 `sum_two(a, b)`를 만들고 실행해보세요.

2. **제곱 구하기 함수**
   - 숫자 하나를 입력받아 그 수의 제곱을 반환하는 함수 `square(n)`을 만드세요.

3. **짝수 판별 함수**
   - 숫자 하나를 입력받아, 짝수면 `True`, 홀수면 `False`를 반환하는 함수 `is_even(n)`을 만드세요.

4. **인사말 함수 (기본값 사용)**
   - 이름을 입력받아 "Hello, [이름]!"을 출력하고, 이름을 입력하지 않으면 "Hello, Guest!"라고 출력하는 함수 `say_hello(name="Guest")`를 작성하세요.

5. **평균 구하기 함수**
   - 숫자 3개(a, b, c)를 받아 평균값을 계산해 반환하는 함수 `get_average(a, b, c)`를 작성하세요.

---

## 5. 자료구조 (05_data_structures.py 관련)

1. **좋아하는 과일 리스트**
   - 좋아하는 과일 5개를 담은 리스트 `fruits`를 만들고, 그 중 3번째 과일을 출력하세요.

2. **리스트 요소 추가와 삭제**
   - 빈 리스트 `nums`를 만들고 숫자 1, 2, 3을 순서대로 추가(`append`)한 뒤, 마지막 숫자를 삭제(`pop` 또는 `del`)하고 리스트를 출력하세요.

3. **딕셔너리 프로필**
   - 본인의 이름, 나이, 취미를 키(key)로 가지는 딕셔너리 `my_profile`을 만들고 출력하세요.

4. **성적 평균 구하기 (리스트 활용)**
   - `scores = [80, 90, 100, 70, 85]` 리스트가 있을 때, 모든 점수의 평균을 구하여 출력하세요. (반복문이나 `sum`, `len` 함수 활용)

5. **중복 제거하기 (집합 Set 활용 힌트)**
   - `numbers = [1, 1, 2, 3, 3, 4, 5]` 리스트가 있습니다. 중복된 숫자를 제거하고 [1, 2, 3, 4, 5]만 남기려면 어떻게 해야 할까요? (힌트: `set()` 자료형을 검색해보거나 사용해보세요)
